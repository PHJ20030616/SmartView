# Task 3.3 LangGraph Resume Parser Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a FastAPI resume parsing endpoint that extracts PDF text, conditionally falls back to PaddleOCR, uses DeepSeek through a LangGraph workflow, and returns contract-validated structured resume JSON.

**Architecture:** A compiled LangGraph `StateGraph` orchestrates deterministic download, PDF extraction, OCR routing, text assembly, DeepSeek JSON extraction, and Pydantic validation nodes. The HTTP endpoint and worker share the same graph runner; Spring Boot, RabbitMQ, and the database remain authoritative for task state and retries.

**Tech Stack:** Python 3.11+, FastAPI, Pydantic Settings, LangGraph, PyMuPDF, PaddleOCR/PaddlePaddle, HTTPX, DeepSeek Chat Completions, pytest.
